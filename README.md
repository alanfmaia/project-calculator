# 🌐 OmniCalc - Suíte de Ferramentas Inteligentes

A **OmniCalc** é uma aplicação web moderna e multifuncional projetada para centralizar ferramentas essenciais de cálculo e conversão em uma interface única, rápida e elegante.

---

## 🚀 Funcionalidades

### 1. 🔢 Calculadora Padrão
*   **Descrição:** Realiza operações matemáticas básicas com precisão.
*   **Recursos:** Adição, subtração, multiplicação, divisão, porcentagem e inversão de sinal (+/-).
*   **Histórico:** Painel lateral que armazena os últimos 10 cálculos realizados para consulta rápida.

### 2. 🧪 Calculadora Científica
*   **Descrição:** Ferramenta avançada para cálculos complexos.
*   **Funções:** Seno (`sin`), Cosseno (`cos`), Tangente (`tan`), Raiz Quadrada (`√`), Logaritmo (`log`), Potência (`xʸ`) e constantes matemáticas (π e e).
*   **Expressões:** Suporte para uso de parênteses para definir precedência de operações.

### 3. ⚖️ Calculadora de IMC
*   **Descrição:** Ferramenta de saúde para cálculo do Índice de Massa Corporal.
*   **Uso:** Insira seu peso (kg) e altura (cm).
*   **Resultado:** Exibe o valor do IMC e a classificação (Abaixo do peso, Ideal, Sobrepeso, Obesidade) com indicadores visuais coloridos.

### 4. 🔄 Conversor de Unidades
*   **Descrição:** Conversor versátil para diversas categorias do dia a dia.
*   **Categorias Suportadas:**
    *   **Comprimento:** Metros, Quilômetros, Centímetros, Milímetros, Polegadas e Pés.
    *   **Peso:** Quilogramas, Gramas, Libras e Onças.
    *   **Temperatura:** Celsius, Fahrenheit e Kelvin.
    *   **Dados (TI):** Bytes, KB, MB, GB, TB e PB (Base 1024).

### 5. 🌐 Calculadora de Sub-rede (Networking)
*   **Descrição:** Ferramenta indispensável para administradores de redes.
*   **Recursos:**
    *   Cálculo baseado em IP e Máscara (CIDR ou Decimal).
    *   **Sincronização:** Escolha o CIDR e a Máscara Decimal atualiza automaticamente (e vice-versa).
    *   **Resultados:** Identifica a Máscara completa, Endereço de Rede, Primeiro Host utilizável, Último Host utilizável, Endereço de Broadcast e o Total de Hosts disponíveis.

## 🛠 Modo de Usar

1.  **Navegação:** Utilize o **menu lateral** (sidebar) para alternar entre as cinco ferramentas disponíveis. Em dispositivos móveis, o menu se desloca para o topo para melhor acessibilidade.
2.  **Entrada de Dados:** 
    *   Na calculadora, utilize os botões na tela ou o **teclado numérico**.
    *   Nas ferramentas de IMC, Conversor e Sub-rede, preencha os campos de texto ou utilize as caixas de seleção.
3.  **Execução:** Clique nos botões de ação (ex: `=`, `Calcular IMC`, `Calcular Sub-rede`) para obter os resultados instantaneamente.
4.  **Limpeza:** O botão **AC** limpa a expressão atual, e o botão de limpar histórico remove os registros da calculadora padrão.
5.  **Histórico:** Na calculadora padrão, clique em um item do histórico para carregar aquele resultado de volta para o visor principal.
